# Build a Deck of Cards and link to html using TypeScript

## My First time using TypeScript in a project

### How to use

- In your terminal type node -v if no version use a google search on how to download & install node.js
- If a Version number is present type node script.js to run
- using the + or - next to number of players and Cards Per Player to increase or decrease.
- Create Deck will build the deck
- Deal will Deal to player1 only at present and display the cards dealt.

#### Future upgrades

- Clear all to clear the cards dealt to player 1 and all players
- Add feature to allow number of players to create the correct number of players and deal to deal cards to them all
- Add rules so you cannot deal more cards than there is avalible
- Ability to play a game of BlackJack
